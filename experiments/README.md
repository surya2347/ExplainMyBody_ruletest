# Experiments

이 폴더는 프로젝트 개발 과정에서 진행한 **실험용 코드와 테스트 파일**을 모아두는 공간입니다.

## 목적

`experiments/` 폴더는 다음과 같은 용도로 사용됩니다.

- 주요 파이프라인에 적용하기 전 아이디어 검증
- OCR 전처리 방법 및 성능 실험
- 규칙 기반 알고리즘 로직 및 임계값 테스트
- LLM 프롬프트 설계 및 응답 비교 실험
- 단일 기능 단위의 간단한 테스트 및 실험

## 참고 사항

- 이 폴더의 코드는 **완성된 코드가 아닐 수 있습니다**
- 실험 목적의 코드로, 구조나 품질이 정제되지 않았을 수 있습니다
- 검증이 완료된 로직만 `src/` 디렉토리로 이동됩니다

핵심 로직과 실험 코드를 분리함으로써  
프로젝트의 안정성과 가독성을 유지하는 것을 목표로 합니다.
